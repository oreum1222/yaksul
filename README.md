# [오름] 국어 학원 약술형 논술 합격 마스터 허브

2027학년도 전국 15개 약술형 논술 실시 대학의 전형 정보 + [오름] 국어 학원 활용 교재(씨에스엠 CSM17) 안내 사이트.

## 특징
- **순수 정적 사이트** — 빌드 도구·프레임워크·서버 없음. HTML/CSS/바닐라 JS.
- 외부 의존성은 Pretendard 폰트(jsdelivr CDN)뿐.
- 다크그린 디자인 시스템 (배경 `#0a0f0c` / 액센트 `#34d399`~`#6ee7a8`).

## 구조
```
index.html              # 메인 허브 (3개 탭: ①대학 비교표(카드) ②대학별 상세 탐색기 ③교재 안내)
{대학}.html              # 대학별 상세 페이지 15개 (gachon, gangnam, kookmin, dongduk,
                        #   sahmyook, sangmyung, seokyeong, suwon, shinhan, eulji,
                        #   kpu, koreatech, hufs, hanshin, hongik_sejong)
files/                  # 대학별 모집요강/가이드 PDF (대학 공식 제공자료)
images/
  oreum-emblem.png      # 헤더 엠블럼 (투명 배경)
  oreum-logo.png        # 가로형 풀로고(엠블럼+워드마크, 현재 미사용·예비)
  og-image.png          # 카카오/SNS 공유 미리보기 이미지 (⚠ 구 디자인 — 교체 필요)
  books/                # 교재 목업 이미지
favicon.png             # 오름 엠블럼
_scratch/               # 작업용 백업·시안 (배포 제외, .vercelignore)
```

- 1·2번 탭의 대학 데이터는 `index.html` 내 `<script>`의 `univData` 배열에 들어 있습니다(수정은 이 배열만 고치면 됩니다).
- 상세 페이지로의 이동/복귀는 `index.html#detail=<파일명>` 해시로 탭·대학 상태를 복원합니다.

## 로컬 실행
빌드 불필요. 정적 서버로 띄우기만 하면 됩니다.
```bash
python -m http.server 8000
# → http://localhost:8000
```
(file:// 로 직접 열면 일부 상대경로/해시 동작이 제한될 수 있으니 로컬 서버 권장)

## 배포
정적 호스팅(Vercel / Netlify / Cloudflare Pages / 일반 웹서버)에 **루트째 업로드**하면 됩니다.
- 빌드 명령: 없음
- Output 디렉토리: 루트(`.`)

### 도메인/공유 이미지 주의
`index.html` `<head>`의 `og:url`, `og:image`, `twitter:image`는 카카오톡 공유 미리보기를 위해 **절대 URL**로 박혀 있습니다. 배포 도메인이 바뀌면 이 3개 값을 새 도메인으로 바꿔 주세요.
